# 🐍 Python Code Explainer

A simple **AI-powered Streamlit web app** that explains Python code in easy, beginner-friendly language.
Users can paste any Python code, and the app will generate a clear explanation using **Google Gemini AI**.
## 🚀 Features
* Paste any **Python code**
* Get a **simple explanation of how the code works**
* Built with **Google Gemini AI**
* Clean and simple **Streamlit interface**
* Helpful for **beginners learning Python**
## 🛠️ Technologies Used
* **Python**
* **Streamlit**
* **Google Gemini API**
* **python-dotenv**
## ⚙️ Installation
1. Clone the repository
```
git clone https://github.com/your-username/python-code-exp.git
```
2. Move into the project folder
```
cd python-code-exp
```
3. Install the required packages
```
pip install -r requirements.txt
```
## 🔑 Setup API Key
Create a **.env** file in the project folder and add your Gemini API key:
```
GEMINI_API_KEY=your
```
## ▶️ Run the App
Start the Streamlit server:
```
streamlit run main.py
```
Your app will open in the browser.
## 💡 Example Use
Input:
```python
for i in range(5):
    print(i)
```
Output:
The app explains that the loop runs 5 times and prints numbers from 0 to 4.
## 🎯 Purpose
This project is designed to help **Python beginners understand code faster** by using AI to break down complex logic into simple explanations.
## 🌐 Deployment
The app can be deployed easily using **Streamlit Cloud** by connecting your GitHub repository.
