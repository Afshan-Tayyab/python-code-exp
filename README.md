# 🐍 Python Code Explainer

A simple **Streamlit web app** that explains Python code using **Google Gemini AI**.
Users can paste Python code and get a clear explanation in simple language.
## 🚀 Features
* Paste Python code
* Get AI-generated explanation
* Simple and beginner-friendly interface
## 🛠️ Technologies
* Python
* Streamlit
* Google Gemini API
